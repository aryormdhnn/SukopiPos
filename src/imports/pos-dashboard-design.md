Design a modern POS (Point of Sale) dashboard interface for a coffee shop, optimized for tablet screens (1024x1366).

The UI should be clean, fast, and touch-friendly, designed for baristas and cashiers working in a busy coffee shop environment.

Style:
Modern, minimal, warm coffee-shop aesthetic. Soft neutral colors (cream, light brown, beige, espresso tones) combined with clean white surfaces. Rounded cards, soft shadows, large touch targets, and clear typography.

Layout Structure:

LEFT SIDEBAR (Navigation)
- Logo / Coffee shop name
- Menu
- Orders
- Transactions
- Inventory
- Customers
- Reports
- Settings

MAIN POS AREA (Center)

Product Categories (horizontal tabs)
- Coffee
- Non Coffee
- Tea
- Pastries
- Snacks

Product Grid
Large product cards optimized for touch:
- product image
- product name
- price
- quick add button

Examples:
- Cappuccino
- Latte
- Americano
- Matcha Latte
- Croissant
- Banana Bread

RIGHT PANEL (Current Order)

Order summary card:
- selected items list
- quantity stepper
- item notes (example: less sugar, oat milk)
- remove item button

Order details:
Subtotal
Tax
Service charge
Total

PAYMENT SECTION

Include local Indonesian payment options:

Digital Payments:
- QRIS
- GoPay
- OVO
- DANA
- ShopeePay

Other:
- Cash
- Debit Card
- Credit Card

Each payment method should be displayed as large touch-friendly buttons with icons.

Extra Features:
- Discount button
- Promo code input
- Split bill
- Print receipt
- Send to kitchen

TOP BAR

- current date & time
- cashier name
- table number (optional for dine-in)
- search product

Interaction Design:

Fast POS workflow:
tap product → add to cart → select payment → confirm payment.

Design should prioritize:
- large touch targets
- clear hierarchy
- fast scanning
- minimal cognitive load

Visual components:
- rounded cards
- clear status colors (success / pending / error)
- smooth micro interactions

Create a realistic POS dashboard UI suitable for a busy coffee shop cashier.
